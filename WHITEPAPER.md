# “迟暮的泪”技术白皮书

版本：0.1.0 (草案)  
最后更新：2026-06-15

## 1. 问题与机会

**问题**：人的记忆会随着时间模糊，物理场景会变迁甚至消失。现有的照片、视频只能被动观看，缺乏沉浸式的“重返”体验。

**机会**：混合现实、4D 场景重建、轻量级脑机接口等技术已从实验室走向消费级。将它们融合，可以创造出首个**情感驱动的空间记忆引擎**。

## 2. 总体架构

系统分为五层三横（详见架构图）。核心模块如下：

### 2.1 采集层
- **消费级采集**：iPhone LiDAR + 全景相机，输出带时间戳的彩色点云流。
- **专业级采集**：无人机 / 机器人 + 工业激光雷达，用于远程或危险环境。
- **情感标记**：可穿戴 ECG/EDA 传感器 + 非侵入式 EEG 头环（如 Muse 2）。

### 2.2 重构层
- **离线重建**：采用 **3D Gaussian Splatting (3DGS)** 或 **4DGS**（支持动态物体）。  
  输入：10分钟扫描视频 → 输出：`.splat` 文件 + 纹理。
- **实时渲染**：基于 WebGL 或 MR 设备原生 SDK（如 HoloLens 2 的 Unreal 插件），达到 30fps 以上。
- **生成式补全**：当场景数据缺失时（如老建筑已拆毁），使用 Stable Diffusion 3 + ControlNet 生成合理的历史风格填充。

### 2.3 交互层
- **无感步态追踪**：腰部 IMU + 深度相机，通过轻量级 LSTM 网络预测用户下一步落点，动态补偿虚拟场景位移。
- **多模态反馈**：
  - 视觉：光波导 MR 眼镜（如 Magic Leap 2）
  - 触觉：超声相控阵（Ultrahaptics）
  - 嗅觉：可编程气味释放（Olorama）
- **情感叙事**：LLM（如 Llama 3 8B 本地部署）根据当前场景和用户生理信号，实时生成个性化旁白。

### 2.4 传输与协同层
- **远程实时重构**：现场采集端通过 5G URLLC 将点云流上传至边缘服务器，服务器 0.5 秒内完成 4DGS 增量重建，推送到客户端。
- **分布式存储**：记忆场景上传到 IPFS，其哈希值锚定在 Arweave 区块链，确保永久可用且不可篡改。

### 2.5 3D 打印接口
- 从高斯泼溅模型中提取选定物体的水密网格（使用 **Gaussian Mesh** 算法），输出 STL/3MF。
- 配合 MR 指导放置：打印后的实体模型可作为空间锚点，增强虚实融合的触感。

## 4. 开放标准与接口

为确保不同模块可以自由组合，我们定义以下开放格式：

- **记忆胶囊**：JSON 文件，包含空间哈希、情感标签、引用场景文件（`.splat` / `.ply`）的 URI。
- **实时传输协议**：基于 QUIC 的二进制流，封装高斯泼溅的梯度更新数据。
- **MR 交互事件**：标准化的触摸、语音、步态事件格式，与 OpenXR 兼容。

## 5. 风险与缓解
