# 🌅 迟暮的泪 (Tears of Nostalgia)

> 让记忆在数字与物理的边界上重新流淌 —— 一个开源的情感空间计算平台。

## 🎯 愿景

**不是冰冷的记录，而是可交互的记忆剧场。**  
每个人都能在任意地点，通过混合现实（MR）技术重现过去的重要时刻 —— 无论是童年的老屋，还是已故亲人的微笑。我们相信，记忆的重构应当成为**全人类共享的数字遗产基础设施**，而非某家公司的封闭产品。

## 🚀 核心能力

- **原地重构**：基于 4D Gaussian Splatting，将手机/LiDAR 扫描的场景在任意空间中1:1重现。
- **情感驱动叙事**：结合脑机接口（可选）与可穿戴传感器，自动调整场景氛围与旁白风格。
- **远程即时同步**：借助 5G/6G 与边缘计算，实时传输千里之外的现场扫描数据。
- **物理触觉延伸**：与 3D 打印结合，让记忆中的物体触手可及。

## 🧩 开放协作架构

本项目采用 **Apache 基金会式治理**，所有技术模块（采集格式、重构算法、交互协议）均采用宽松开源许可证。欢迎任何个人、机构、企业以以下方式参与：

- **代码贡献**：修复 bug、开发新模块、编写文档。
- **硬件捐赠**：提供 MR 设备、传感器、3D 打印机等。
- **场景测试**：养老院、医院、博物馆等实地验证。
- **伦理监督**：加入独立道德委员会，确保技术向善。

## 📖 快速开始

### 1. 体验 WebXR 原型（无需任何设备）
打开手机（iOS/Android），访问 [getting-started/index.html](getting-started/index.html)（需部署到 HTTPS 服务器）。  
在任意地点点击“录制记忆”，然后离开再回来 —— 你将听到自己录制的语音自动播放。

### 2. 构建与运行完整版（需要 MR 设备）
请参阅 [WHITEPAPER.md](WHITEPAPER.md) 中的开发环境配置。
