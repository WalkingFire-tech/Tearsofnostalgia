# 伦理框架与技术向善承诺

“迟暮的泪”项目不仅是一套技术方案，更是一份**人文契约**。所有贡献者、使用者和下游提供方必须遵守本框架。

## 1. 核心原则

1. **记忆主权**：用户的记忆数据永远归用户所有，任何云端服务需获得明确授权且可随时删除。
2. **透明可解释**：情感分析、生成式补全等 AI 模块必须提供其判断依据（例如热力图、置信度）。
3. **禁止恶意使用**：不得用于制造虚假记忆、情绪操控、身份盗窃或任何形式的欺诈。
4. **脆弱人群保护**：涉及儿童、老年人、认知障碍患者的应用，必须由伦理委员会审批。

## 2. 脑机接口（BCI）特殊条款

鉴于 BCI 可以读取甚至影响神经活动，对使用 BCI 的模块做以下强制约束：

- **物理开关**：任何 BCI 设备必须配备硬件级开关，用户可以随时切断信号采集。
- **数据本地化**：原始 EEG 信号不允许离开用户的本地网络，仅输出脱敏后的情感标签（如“兴奋度 0.7”）。
- **禁止无感诱导**：不得利用 BCI 读取到的情绪状态，在用户未明确同意的情况下推送广告或内容。
- **记忆安全**：禁止任何形式的“记忆写入”或“记忆擦除”功能。MR 场景中的虚拟物体必须明确标记为“合成内容”。

## 3. 伦理审查流程

- **项目级审查**：每个主要版本发布前，由独立道德委员会（至少 3 名成员，包括伦理学者、法律专家、用户代表）进行风险评估。
- **应用级审查**：第三方基于本项目开发的应用，如需接入 BCI 或涉及敏感人群，应自行向当地伦理委员会申请批准，并向项目组备案。

## 4. 违规处理

任何违反本框架的代码提交、发行版或商业应用，项目组保留**公开谴责、撤销开源授权、配合法律调查**的权利。
